# Task Manager React 

A React-based application for managing tasks efficiently.

## Features
- Add, edit, and delete tasks
- Responsive and user-friendly interface
- Powered by React 18

## Project Structure
- **`src/`**: Contains the source code for the app.
- **`public/`**: Holds static assets and the entry `index.html`.
- **`node_modules/`**: Stores project dependencies.
- **`package.json`**: Project metadata and dependency definitions.

## Installation

1. Clone or download the repository.
2. Navigate to the project directory.
3. Install dependencies using: npm install


## Usage

### Development
To run the app in development mode: npm start

Open [http://localhost:3000](http://localhost:3000) in your browser to view the application.



### Build
To build the app for production: npm run build


The build files will be in the `build/` directory.

## Dependencies
- **React**: ^18.0.0
- **React DOM**: ^18.0.0
- **React Scripts**: 5.0.1

## Browsers Support
- **Production**: Modern browsers (>0.2%, not dead, not Opera Mini)
- **Development**: Latest versions of Chrome, Firefox, and Safari
